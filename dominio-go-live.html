Go-live

Domínio, SEO e validação final.

Use amanhã, após a Vercel liberar, para conferir se o domínio, DNS, Supabase, sitemap e páginas públicas estão prontos para divulgação.

Passo a passo no Vercel

  1. Abra Project arandu → Settings → Domains.
  2. Adicione o domínio comprado, com e sem www.
  3. Use o DNS recomendado pela Vercel.
  4. Defina o domínio principal como production.
  5. Confirme HTTPS ativo.
  6. Atualize ARANDU_SITE_URL para o domínio final.
