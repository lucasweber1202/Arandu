Privacidade

Como a Arandu trata dados no MVP.

Esta é uma política inicial para o MVP estático. Ela deve ser revisada juridicamente antes da operação comercial definitiva.

Dados coletados

Podemos coletar nome, e-mail, WhatsApp, briefing de projeto, mensagem de contato e portfólio enviado voluntariamente por formulários.

Uso dos dados

Os dados são usados para atendimento curatorial, resposta a compradores, análise de artistas e relacionamento com arquitetos e empresas.

Armazenamento atual

No MVP, formulários usam armazenamento local no navegador quando ativados. Em produção, os dados deverão ir para serviço seguro.

Compartilhamento

Dados não devem ser vendidos. Podem ser compartilhados apenas quando necessário para atendimento, entrega, proposta ou documentação.

Direitos do usuário

O usuário poderá solicitar correção ou remoção de dados por meio do contato oficial da Arandu.

Contato

Dúvidas sobre privacidade devem ser enviadas para o canal oficial da curadoria.
