# Seguranca e privacidade

## Dados coletados no MVP

- Nome.
- Email.
- Telefone.
- Mensagem.
- Interesse em obra.

- Portfolio de artista.

## Regras

- Coletar apenas dados necessarios.
- Nao pedir documentos sensiveis no MVP.
- Nao expor leads em areas publicas.
- Separar chaves privadas de codigo.
- Usar variaveis de ambiente quando houver integracao com banco.
