# Checklist comercial — parceria com artistas

Este arquivo organiza os pontos que a Arandu deve alinhar com cada artista antes de publicar obras. Não é contrato final. Deve servir como base para conversa comercial e posterior revisão jurídica/contábil.

## 1. Dados do artista

- Nome artístico.
- Nome completo.
- Cidade e estado.
- E-mail.
- WhatsApp.
- Instagram.
- Portfólio.
- Linguagens principais.
- Mini bio.
- Breve trajetória.
- Declaração ou texto do artista.

## 2. Dados mínimos de cada obra

- Título.
- Ano.
- Técnica.
- Suporte.
- Dimensões.
- Edição ou tiragem, quando houver.
- Preço ou faixa de preço.
- Status: disponível, reservada, vendida ou não publicada.
- Fotos autorizadas.
- Observações de conservação.
- Moldura, base ou acabamento.
- Cidade onde a obra está localizada.

## 3. Autoria e procedência

Confirmar com o artista:

- se a obra é de autoria própria;
- se a obra pode ser comercializada;
- se há alguma restrição de imagem;
- se a obra já foi exposta, publicada ou vendida anteriormente;
- se existe certificado próprio do artista;
- se há documentação complementar.

## 4. Condições comerciais

Definir antes da publicação:

- comissão Arandu;
- valor líquido esperado pelo artista;
- margem para negociação;
- possibilidade de desconto;
- prazo de pagamento ao artista;
- se a obra fica ou não em exclusividade;
- prazo máximo de reserva;
- procedimento caso a obra seja vendida por outro canal.

## 5. Logística

Definir por obra:

- quem embala;
- quem paga a embalagem;
- quem calcula o frete;
- quem contrata o transporte;
- se haverá seguro;
- prazo estimado de envio;
- responsabilidade em caso de dano;
- necessidade de retirada presencial.

## 6. Certificado

Confirmar:

- se a obra terá certificado Arandu;
- se o artista também emitirá certificado próprio;
- quais informações entram no certificado;
- se haverá assinatura física ou digital;
- se o código de verificação será público;
- se a obra exige laudo externo ou não.

## 7. Uso de imagem e divulgação

Alinhar autorização para uso das imagens em:

- página da obra;
- página do artista;
- redes sociais;
- proposta para compradores;
- material institucional;
- apresentações para arquitetos e empresas.

## 8. Sinais de alerta

Não publicar obra quando houver:

- dúvida sobre autoria;
- imagem sem autorização;
- preço indefinido;
- status incerto;
- dados técnicos incompletos;
- divergência sobre comissão;
- dúvida sobre frete ou responsabilidade de envio;
- obra indisponível ou já comprometida.

## 9. Decisão final de publicação

Antes de publicar, marcar:

- [ ] Dados do artista completos.
- [ ] Dados da obra completos.
- [ ] Fotos aprovadas.
- [ ] Preço validado.
- [ ] Comissão validada.
- [ ] Logística definida.
- [ ] Certificado definido.
- [ ] Uso de imagem autorizado.
- [ ] Status de disponibilidade confirmado.
