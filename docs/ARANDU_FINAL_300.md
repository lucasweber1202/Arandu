# Arandu — Pacote Final 300

Este documento registra o pacote final de melhorias implementado no fechamento do dia. O pacote foi estruturado para funcionar como uma camada global de produto, curadoria, conversão, acessibilidade e otimização, com foco em preview via Codespace e posterior deploy único no Vercel.

- `js/arandu-final-300.js`
- `css/arandu-final-300.css`

- `index.html`

- `js/site.js`

## Principais entregas funcionais

- Command palette com atalhos de navegação e busca.
- Mapa de intenções curatoriais.
- Perfis de comprador.
- Templates de mensagem para contato e orientação.
- FAQ curatorial dinâmica.
- Helper de orçamento.
- Cópia da jornada do usuário.
- Auditoria local de links, botões, formulários e seções.
- Memória de rolagem por página.
- Proteção de links vazios.
- Ajustes automáticos de acessibilidade.
- Hints de lazy loading para imagens.
- Selo de confiança persistente.
- Prompt de saída.
- Indicadores de seções.
- Registro local de eventos.
- Integração com intenção, seleção e jornada.

## Objetivo do pacote

Aumentar a sensação de galeria curatorial, reduzir fricção de compra, melhorar orientação do usuário, fortalecer confiança e preparar o site para um deploy único mais robusto no Vercel.
