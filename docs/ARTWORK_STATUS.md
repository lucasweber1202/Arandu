# Matriz de status das obras — Arandu

## available

Obra disponível para venda ou proposta curatorial.

## reserved

Obra temporariamente reservada para comprador, arquiteto ou projeto. Deve ter prazo claro.

## sold

Obra vendida. Deve ter certificado emitido, dados de venda registrados e status bloqueado para nova venda.

## archived

Obra fora do acervo comercial. Pode permanecer no histórico do artista, mas não deve aparecer como disponível.

## under_review

Obra enviada pelo artista, mas ainda em análise curatorial, técnica ou documental.

## pending_certificate

Obra vendida ou em processo de venda, aguardando emissão de certificado.

## pending_documentation

Obra com dados incompletos: técnica, dimensão, imagem, ano, preço, autoria ou disponibilidade.

## exhibition_only

Obra exibida para fins curatoriais ou editoriais, sem disponibilidade comercial imediata.
