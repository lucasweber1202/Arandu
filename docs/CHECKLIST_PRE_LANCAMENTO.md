# Checklist de pré-lançamento — Arandu

## Site

- Home explica a Arandu em poucos segundos.
- Página de obras ativa.
- Página de artistas ativa.
- Página de curadoria ativa.
- Página de confiança/autenticidade ativa.
- Página para artistas ativa.
- Página para compradores ativa.
- Página de contato ativa.
- Login e cadastro disponíveis.

## Captação

- Formulário de artistas funcionando.
- Formulário de compradores funcionando.
- Formulário de contato funcionando.

- Catálogo inicial revisado.
- Critérios curatoriais claros.
- Fluxo de compra assistida explicado.
- Certificado e autenticidade explicados.
- Política comercial preliminar revisada.

- Testar cadastro e login.

- Testar painel.
