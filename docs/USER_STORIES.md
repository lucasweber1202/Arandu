# User stories

## Comprador

Como comprador, quero encontrar obras por ambiente para nao depender de conhecimento tecnico sobre arte.

Como comprador, quero salvar obras em uma selecao para pedir orientacao antes de decidir.

Como comprador, quero entender a escala da obra para saber se ela funciona no meu espaco.

## Artista

Como artista, quero enviar meu portfolio para avaliacao curatorial.

Como artista, quero que minha pagina apresente minha trajetoria, nao apenas obras isoladas.

## Arquiteto

Como arquiteto, quero enviar briefing de projeto para receber uma selecao coerente com ambiente, prazo e orcamento.

Como arquiteto, quero fichas tecnicas claras para apresentar obras ao cliente.
