# Briefings

A Arandu deve coletar informacoes suficientes para orientar a escolha de obras.

Comprador: nome, contato, ambiente, momento, orcamento e mensagem.

Artista: nome, cidade, linguagem, portfolio e obras disponiveis.

Projeto: responsavel, escritorio, ambiente, prazo, orcamento, medidas e referencias.

A experiencia deve soar como inicio de curadoria acompanhada.
