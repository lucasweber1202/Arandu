# Guia de imagens — Arandu

## Objetivo

Padronizar imagens de obras, artistas e compartilhamento antes de iniciar prospecção real.

## Imagens de obras

Capturar sempre que possível:

1. Obra frontal, sem distorção.
2. Detalhe de textura, material ou acabamento.
3. Obra no ambiente ou simulação de escala.
4. Verso, assinatura ou detalhe documental quando relevante.

## Padrões técnicos

- Largura mínima recomendada: 1600 px.
- Formato preferencial: JPG ou WebP.
- Fundo neutro.
- Boa iluminação.
- Sem filtros exagerados.
- Sem marca d'água sobre a obra.

## Nome de arquivo

```text
nome-da-obra-nome-do-artista.jpg
```

Exemplo:

```text
estudo-de-solo-04-marina-silveira.jpg
```

## Imagens de artistas

Preferir fotos que mostrem:

- artista no ateliê;
- relação com matéria, processo ou obra;
- retrato simples para perfil.

## Alt text

O alt text deve descrever a imagem de forma objetiva:

```text
Obra Estudo de Solo Nº 04, pintura em tons terrosos de Marina Silveira.
```

Evitar:

```text
Imagem bonita de arte.
```

## Imagens para redes sociais

Criar versões horizontais para compartilhamento:

- 1200 x 630 px para Open Graph.
- 1080 x 1350 px para Instagram feed.
- 1080 x 1920 px para stories.

## Compressão

Antes do deploy, comprimir imagens para evitar site pesado. Preferir WebP em produção quando possível.
