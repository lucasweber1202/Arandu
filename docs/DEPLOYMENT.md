# Deploy

## Fase MVP

A primeira versao pode ser publicada em Vercel, Netlify ou outro servico estatico compatível com Vite.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Variaveis futuras

Quando Supabase for conectado, criar variaveis para URL do projeto e chave publica anonima.

## Dominio

Usar dominio proprio apenas quando a home, os formularios e a jornada Encontre arte para estiverem minimamente estaveis.

## Criterio para publicar

- Logo correta.
- Home clara.
- Curadoria por contexto visivel.
- Formularios funcionais.
- Mobile aceitavel.
