# Arandu — checklist de lançamento e redes sociais

## 1. Conferência técnica

Rodar na main:

```bash
git checkout main
git pull
git status
npm run check:quality
npm run check:all
npm run build
npm run preview -- --host 0.0.0.0
```

Critérios:

- checks sem erro crítico;
- build finalizado;
- preview aberto;
- working tree limpo.

## 2. Conferência visual

Revisar:

```text
/
login.html
cadastro.html
minha-conta.html
portal-artista.html
empresas.html
comprar-arte.html
obras.html
obra.html?id=estudo-de-solo-04
confianca.html
para-artistas.html
```

Verificar:

- botão Entrar visível no header;
- login com comprador, artista e empresa;
- home sem texto claro em fundo claro;
- ausência de blocos duplicados;
- cards sem sobreposição;
- tags, preços e títulos legíveis;
- empresas com briefing rápido;
- portal do artista com reavaliação de preço.

## 3. Pendências de produção
