Uso interno

Painel para operar o Arandu antes de divulgar.

Aqui você acompanha submissões, leads, obras, artistas, reservas, certificados, briefs, propostas e tarefas. A página usa a API consolidada em /api/admin e exige o ARANDU_ADMIN_TOKEN.

Fluxo mínimo

Submissão → curadoria → cadastro → certificado → reserva

O primeiro lançamento não precisa de checkout completo. Precisa de controle, confiança e resposta rápida para artistas e compradores.

Acesso

Token administrativo

Use o mesmo valor configurado na Vercel como ARANDU_ADMIN_TOKEN. Ele fica salvo apenas neste navegador.

Setup produção

Importante: se o Supabase ainda não estiver configurado, a API responde em modo demo. Depois das variáveis reais, o painel passa a ler e gravar no banco.
