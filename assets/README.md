# Assets — Arandu

Estrutura recomendada para arquivos visuais da Arandu.

## Pastas sugeridas

```text
assets/logo/
assets/obras/
assets/artistas/
assets/social/
assets/certificados/
```

Como o Git não versiona pastas vazias, cada pasta tem um `.gitkeep`.

## Logo

Arquivo recomendado:

```text
assets/logo-arandu.png
```

Também manter versão vetorial quando possível:

```text
assets/logo-arandu.svg
```

## Imagens de obras

Padrão de nome:

```text
assets/obras/nome-da-obra-artista.jpg
```

Tamanhos recomendados:

- largura mínima: 1600 px;
- formato: JPG para foto, PNG para transparência, WebP para performance;
- variações: frontal, detalhe, ambiente.

## Imagens de artistas

Padrão de nome:

```text
assets/artistas/nome-do-artista.jpg
```

## Social

Usar para imagens de compartilhamento:

```text
assets/social/og-home.jpg
assets/social/og-obras.jpg
assets/social/og-artistas.jpg
```

## Certificados

Usar para PDFs ou imagens de certificados emitidos:

```text
assets/certificados/ARD-2026-0001.pdf
```
